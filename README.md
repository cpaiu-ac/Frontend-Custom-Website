# Custom Front-End Website Implementation

## 📌 About the Project
This repository contains the source code for a custom, multi-page website built entirely from scratch. This project serves as a practical demonstration of my front-end development capabilities, validating the web management and UI structure improvement experience highlighted in my professional background.

## ⚙️ Core Features
* **Multi-Page Architecture:** Includes several interconnected HTML pages (e.g., `index.html`, `index2.html`) allowing for logical content distribution and seamless navigation.
* **Custom Styling:** Utilizes a dedicated external stylesheet (`main.css`) to maintain consistent layout, typography, and visual hierarchy across all pages.
* **Clean Code Structure:** Emphasizes semantic HTML5 and well-organized CSS rules for maintainability and scalability.

## 🛠️ Tech Stack & Concepts
* **Languages:** HTML5, CSS3
* **Concepts:** UI/UX Design principles, Front-End Architecture, DOM Structure.

## 📁 Repository Structure
* `index.html`, `index2.html`, `index-2.html` - The core structural markup pages of the website.
* `main.css` - The primary stylesheet dictating the visual presentation.
